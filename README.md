# Ether Transfer Smart Contract

## Deployed on SCAI Mainnet
- Network: SecureChain Mainnet (Chain ID: 34)
- Contract Address: 0xb57025117ed4cC9882E4e8EA492953C0597383EB
- EVM Version: london

## Functions
- deposit() - Send SCAI to contract
- sendEther() - Transfer SCAI to any address  
- withdraw() - Owner withdraws all SCAI
- getBalance() - Check contract balance
